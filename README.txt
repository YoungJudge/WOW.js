#使用方法
1.引入animate动画集CSS插件--animate.css
2.javascript中：new WOW().init();
3.使用对象添加class="wow [animate名称]"
4.可用属性：
	data-wow-duration(动画持续时间，单位s)
	data-wow-delay(动画延迟时间，单位s)
	data-wow-offset(偏移量，无单位，代表距离可视区域多远开始执行动画)
	data-wow-iteration="10"(重复次数)
5.animate.css的动画类名详见：http://daneden.github.io/animate.css/
